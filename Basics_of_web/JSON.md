## JSON
人間でも読んだり、作ったりすることができるように考えられたデータフォーマット


### JSON特徴

- データ量が軽い

- 言語に依存しない

- 読みやすい構造

### JSONの記述方法
{}で囲む
キー(key)と値は(value)をコロンで区切って格納
2で作成したデータをそれぞれカンマで区切る

### JSONとXMLの違い
XML

- データとし文字列しか表す事ができない

- データサイズが大きくなりがち

JSON

- 数値や空も表現できる

- データファイルは小さめ

XMLのようにDOMを使う必要がなく軽量であるためJSONが使われる事が多い
